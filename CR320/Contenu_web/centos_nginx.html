Informations du Conteneur CentOS

Type de Conteneur : CentOS

Serveur Web Utilisé : Nginx

Comment installer Nginx sur CentOS ?

Pour installer Nginx sur CentOS, utilisez la commande suivante :

sudo yum install -y epel-release && sudo yum install -y nginx

Comment ajouter une page web ?

Placez vos fichiers HTML dans le répertoire suivant :

/usr/share/nginx/html/
