Informations du Conteneur Ubuntu

Type de Conteneur : Ubuntu

Serveur Web Utilisé : Apache (httpd)

Comment installer Apache (httpd) sur Ubuntu ?

Pour installer Apache sur Ubuntu, utilisez la commande suivante :

sudo apt-get update && sudo apt-get install -y apache2

Comment ajouter une page web ?

Placez vos fichiers HTML dans le répertoire suivant :

/var/www/html/
